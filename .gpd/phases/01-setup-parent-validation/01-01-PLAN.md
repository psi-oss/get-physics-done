<task type="auto">
  <name>Task 3: Reproduce SU(2) Ward identity benchmarks</name>
  <files>benchmarks/su2_ward_identity.dat</files>
  <action>
  1. Set up production runs for SU(2) N=4 SYM on 4^4 A_4* lattice
  2. Run at 2+ coupling values spanning the range used in Catterall-Schaich (arXiv:1505.03135):
     - Moderate coupling: lambda_lat ~ 1.0
     - Weaker coupling: lambda_lat ~ 0.5
  3. For each coupling:
     - Thermalize: >= 500 RHMC trajectories (monitor bosonic action for plateau)
     - Measure: >= 1000 configurations (every 5-10 trajectories)
     - Compute: bosonic action per site <S_B/N_site>, statistical error via jackknife
  4. Compare measured <S_B/N_site> to published values in arXiv:1505.03135
  5. Exact SUSY prediction for SU(2): <S_B/N_site> = (9/2)(4) - 1/2 = 17.5 (in the continuum limit)
  6. Record all parameters and results in benchmarks/su2_ward_identity.dat

  DIMENSIONAL CHECK: <S_B/N_site> is dimensionless (action per site). Expected value O(10) for SU(2).
  </action>
  <verify>
  - Ward identity value agrees with published result within 2-sigma
  - Statistical errors are well-estimated (jackknife vs bootstrap comparison)
  - Value approaches 17.5 as coupling weakens (continuum limit direction)
  - Thermalization is verified: running average of S_B shows clear plateau before measurements begin
  </verify>
  <done>Ward identity <S_B/N_site> matches Catterall-Schaich at 2+ couplings within 2-sigma</done>
</task>

<task type="auto">
  <name>Task 4: Verify Pfaffian phase for SU(2)</name>
  <files>benchmarks/su2_pfaffian_phase.dat</files>
  <action>
  1. On 2^4 lattice with SU(2), generate 100+ thermalized configurations
  2. For each configuration, compute the exact Pfaffian of the fermion operator M:
     - Construct the full antisymmetric matrix M (dimension: 16 * N_c^2 * V = 16 * 4 * 16 = 1024)
     - Compute Pf(M) using Parlett-Reid tridiagonalization (Python pfaffian library)
     - Record both |Pf(M)| and arg(Pf(M))
  3. For SU(2) N=4 SYM, the Pfaffian is known to be real and positive (Pf > 0)
  4. Compute: histogram of arg(Pf), mean <cos(alpha)>, fraction of configs with Pf < 0
  5. Record results in benchmarks/su2_pfaffian_phase.dat

  DIMENSIONAL CHECK: Pfaffian is dimensionless in lattice units. |Pf| ~ exp(O(V)) is expected.
  </action>
  <verify>
  - All Pfaffians have arg(Pf) = 0 within numerical precision (real positive)
  - <cos(alpha)> = 1.000 +/- epsilon with epsilon < 0.01
  - No configurations with Pf < 0
  - This matches the known result for SU(2) (Catterall-Schaich, arXiv:1505.03135)
  </verify>
  <done>Pfaffian confirmed real positive for SU(2) on 2^4 lattice (100+ configs, all arg(Pf) = 0)</done>
</task>
