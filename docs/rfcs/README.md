# RFCs

This directory holds dated design notes for proposed or transitional work.

Use it for:

- scope-lock RFCs
- rollout and migration notes tied to an active proposal
- dated design records that are not yet implemented enough to become shipped references

Do not use it for evergreen runtime reference material.

Shipped, implementation-aligned reference docs belong under `src/gpd/specs/references/`.
When a feature becomes real and stable, add or update an evergreen reference there and
leave the dated proposal history here.
